Novice's Guide to Basic GPU Overclocking

NVIDIA Overclocking Rules

  • Core Clock Offset (MHz): Enter values like -200 or +150 to offset the graphics clock.
  • Absolute Core Clock (MHz): Enter a value above 500 (e.g. 1450) to lock the core clock to that exact frequency. This reduces heat and power consumption. (Recommended for RTX cards).
  • Memory Clock Offset (MHz): In HiveOS (Linux), memory clocks are doubled compared to Windows. E.g., if you set +1000 in Windows, enter 2000 in HiveOS.
  • Power Limit (Watts): Limit the maximum power. Setting it to 0 or leaving it blank disables limits (allowing maximum base spec).

AMD Overclocking Rules

  • Core Clock (MHz) & Voltage (mV): Set the exact core frequency and target core voltage (e.g., 1150 MHz at 800 mV).
  • Memory Clock (MHz): Set the memory speed directly (e.g. 2000).
  • MVDD & VDDCI: Core memory and controller voltages (set to 0 to use defaults if you are unsure).
  • DPM (State): Power states 1-7. (Set to 4 or 5 for stable power control on Polaris/Vega cards).
